# Design System — CCC Le Zoo (Cadre minimal)

Objectif: fournir un cadre, sans choisir d'apparence définitive, afin que le Web Designer prenne des décisions graphiques cohérentes.

## Couleurs (variables uniquement)

- --cccz-primary: #1a5a4a; /*institution placeholder*/
- --cccz-accent: #f0b429; /*accent placeholder*/
- --muted: #666;

> Ces valeurs sont des placeholders pour référence, NE PAS les considérer comme choix définitifs.

## Typographie recommandée

- Famille système: Inter, system-ui, -apple-system, 'Segoe UI', Roboto, 'Helvetica Neue', Arial
- Titres: graisse semi-bold à bold
- Corps de texte: 16px base, hauteur de ligne 1.4-1.6

## Règles de sobriété

- Utiliser peu de couleurs distinctes
- Contraste suffisant entre texte et fond
- Marges et paddings modérés

## Hiérarchie visuelle attendue

- H1: très lisible, H2 section, H3 sous-section
- Les CTA doivent être clairement indentifiables mais non agressifs

## Principes UX

- Lisibilité prioritaire (contraste, taille, interlignage)
- Navigation claire et prévisible
- Accessibilité: alt pour images, labels pour inputs, structure sémantique

## Composants

- Fournir composants réutilisables simples (Button, SectionTitle, Card) sans styles définitifs

## Documentation

- Chaque décision graphique finale doit être documentée dans le guide du designer.
